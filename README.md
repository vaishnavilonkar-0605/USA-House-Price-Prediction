# 🏡 USA House Price Prediction

A Machine Learning project that predicts house prices in the USA using regression techniques and data analysis. This project demonstrates the complete ML workflow including data preprocessing, exploratory data analysis (EDA), model training, prediction, and evaluation using Python and Scikit-learn.

---

# 📌 Project Overview

The goal of this project is to build a predictive model that estimates house prices based on various features such as:

- Average Area Income
- Average Area House Age
- Average Area Number of Rooms
- Average Area Number of Bedrooms
- Area Population

Using these features, the model predicts the final house price with the help of **Linear Regression**.

This project is beginner-friendly and helps in understanding:
- Supervised Machine Learning
- Regression Algorithms
- Model Evaluation Metrics
- Data Visualization
- Real-world prediction problems

---

# 📂 Dataset Information

The dataset used in this project contains housing data from different regions in the USA.

## Features

| Column Name | Description |
|---|---|
| Avg. Area Income | Average income of residents |
| Avg. Area House Age | Average age of houses |
| Avg. Area Number of Rooms | Average number of rooms |
| Avg. Area Number of Bedrooms | Average number of bedrooms |
| Area Population | Population of the area |
| Price | Target variable (House Price) |
| Address | Address of the house |

# 📈 Model Performance

The model performance was evaluated using regression metrics:

- **MAE** → Measures average prediction error
- **MSE** → Penalizes larger errors
- **RMSE** → Gives error in understandable units

Lower values indicate better model performance.
